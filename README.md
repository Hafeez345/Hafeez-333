🤖 AI Data Analysis Agent

An intelligent AI-powered data analysis agent built using Python and Streamlit.
This agent understands natural language questions (English & Urdu) and performs accurate data analysis on employee datasets.

🚀 Features

Natural Language Question Understanding

Intent-based logic system

Exact match filtering (e.g., salary exactly 100000)

Highest / Lowest / Top N queries

Data visualization with charts

Smart number and name extraction

CSV dataset support

🧠 How It Works

User asks a question

System detects intent

Extracts name/number/state

Applies Pandas logic

Returns exact results (table or chart)

📊 Supported Query Types

Salary by employee name

Contact number by name

State of employee

Rank of employee

Employee ID by name

Highest salary

Lowest salary

Top 3 employees

Employees earning exactly X amount

Average salary

🛠️ Technologies Used

Python

Streamlit

Pandas

Plotly

JSON-based intent engine
